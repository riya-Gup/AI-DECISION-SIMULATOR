# 🧠 AI Decision Simulator (Monte Carlo) 📊

## 📌 Project Overview

This project presents an interactive **Monte Carlo‑based decision simulator** for major life choices – job switch, higher studies, startup, or moving to a new city. Instead of single‑point advice, it runs **10,000 simulations** to show probability distributions, risk metrics, and comparative outcomes.

Using an intuitive web interface and dynamic visualizations, users can explore:

- **Probability** that an action (switch/study/startup/move) beats staying
- **Mean wealth** and **10th percentile (risk)** over 5 years
- **Overlapping histograms** comparing “Stay” vs “Action”
- **Scenario‑specific inputs** (salary, growth, layoff risk, tuition, loan interest, success rate, cost of living, etc.)
- **Rule‑based insights** based on risk tolerance

## ⚙️ Features

- 🎲 **Monte Carlo Engine** – 10,000 random futures per decision  
- 📈 **Interactive Histograms** (Plotly.js) – overlay distributions  
- 🧮 **Four decision scenarios** – Job Switch, Higher Studies, Startup, Move City  
- 🧠 **Risk awareness** – 10th percentile, volatility, probability of better outcome  
- 🎨 **Glass‑morphism UI** – modern, responsive, centered layout  
- 🧪 **No backend or installation** – runs entirely in the browser  

## 🛠️ Tools & Technologies

- **HTML5 / CSS3** – structure & styling (glass‑morphism, flexbox)  
- **JavaScript (ES6)** – Monte Carlo logic, UI controllers  
- **Plotly.js** – interactive histogram visualizations  
- **JSON** – external data for industry & macro parameters  

## 🚀 How to Use

1. **Download or clone** this repository  
2. Open `index.html` in any modern browser (Chrome, Edge, Firefox)  
3. Select a decision type from the dropdown  
4. Adjust input parameters (salaries, percentages, sliders)  
5. Click **📊 Calculate & Simulate**  
6. View probability, wealth metrics, and the interactive histogram  

> *No Power BI or database required – it’s a pure static web app.*
